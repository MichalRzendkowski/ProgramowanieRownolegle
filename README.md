# ProgramowanieRownolegle
Repozytorium z zdaniami z Programowania Równoległego i Rozproszonego

## Java
Celem zadania jest stworzenie aplikacji wielowątkowej w Java do obsługi sieci taksówek.
Folder zawiera dwie implementacje klasy DyspozytorniaWatkowa.

## MPI
Celem zadania jest zrównleglenie obliczeń fizycznych. Program oblicza siły i orientację 3-wymiarowej kostki magnesów. Wymiary siatki są dowolne.
